# Prompts
Code for project about prompt effects in learner corpora and production

Please cite this repository with the following:
Hernández, A., Minnillo, S., Fernández-Mira, P. & Sánchez-Gutiérrez, C. (2026, March). Lexis, fluency, and tense in oral narrations: comparing the effect of valence across prompts. Oral Presentation at _American Association of Applied Linguistics Conference_.

To learn more about the project, please see this publication:
Sánchez-Gutiérrez, C., Minnillo, S., Fernández-Mira, P. & Hernández, A. (2024). Prompt response variation in learner corpus research: Implications for data interpretation. _Research Methods in Applied Linguistics, 3_(3). https://doi.org/10.1016/j.rmal.2024.100134
